# FB 自動封存舊貼文

一個自動將 Facebook 個人頁面舊貼文移到儲藏盒的 Chrome 瀏覽器擴充功能。

## 功能特色

- 自動偵測並封存超過指定天數的貼文
- 支援自訂天數（1-365 天）
- 自己發的貼文 → 移到儲藏盒
- 好友標記的貼文 → 從個人檔案隱藏
- 即時顯示處理進度（已封存、略過、失敗數量）
- 可隨時停止操作

## 安裝方式

1. 下載本專案或 Clone 下來
2. 打開 Chrome，輸入 `chrome://extensions`
3. 開啟右上角「開發者模式」
4. 點選「載入未封裝擴充功能」
5. 選擇本專案資料夾

## 使用說明

1. 進入 Chrome 擴充功能，點擊本擴充功能圖標
2. 設定要封存的天數
3. 先前往自己的 Facebook 個人頁面 (`https://www.facebook.com/me`)
4. 按下「開始封存舊貼文」
5. 等待自動處理完成

## 注意事項

- 操作無法復原，被封存的貼文可在 Facebook 儲藏盒中找回
- 請確保在個人頁面使用，勿在其他頁面使用
- 處理速度已設為適當延遲，避免被 Facebook 偵測為自動化操作

## 技術栈

- Manifest V3
- Vanilla JavaScript
- Chrome Extensions API
